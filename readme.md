# Minha API

Este é um projeto de demonstração para uma API simples de gerenciamento de produtos, desenvolvido com Spring Boot.

## Descrição

A API permite cadastrar, listar, atualizar e deletar produtos.

## Endpoints

### Cadastrar Produto


Cria um novo produto com base nos dados fornecidos no corpo da requisição.

### Listar Produtos


Retorna uma lista de todos os produtos cadastrados.

### Deletar Produto


Remove o produto com o ID especificado.

### Atualizar Produto



Atualiza os dados do produto com o ID especificado, com base nos dados fornecidos no corpo da requisição.

## Tecnologias Utilizadas

- Spring Boot
- Java

## Como Executar

1. Certifique-se de ter o Java JDK e o Maven instalados.
2. Clone este repositório.
3. Navegue até o diretório do projeto.
4. Execute o comando `mvn spring-boot:run` para iniciar o aplicativo.
